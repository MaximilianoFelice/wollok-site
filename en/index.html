What is Wollok?

Wollok is not only a programming language but also an IDE (Integrated Development Environment) specially designed for teaching initial OOP courses. It is a great tool to incrementally add every OO concept, allowing the students to program from the beginning!

How is Wollok language?

  • Object-oriented
  • Classes are an optional abstraction, objects are first-class citizens. You can program thinking about objects and then introduce new concepts like Packages, Inheritance, Overloading, etc.
  • Has a clear and simple syntax, easy to understand and use.
  • It is a dynamic language, without types annotations...
  • ... but has an implicit type system, so, although dynamic, it has several compile-time checkings
  • It is interpreted.

  How is the Integrated Development Environment?

  • Fully integrated to Eclipse and its plugins.
  • Checks and Quick Fixes like any other Eclipse platform.
  • It has a compile-time error report
  • With a REPL Integrated Interactive Console.
  • With own Debugger tool
  • With Static & Dynamic diagrams.
  • With a Graphic Environment for small games (work in progress)
